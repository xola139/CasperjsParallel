# CasperjsParallel
Simple ejemplo Scraping utilizando Casperjs, GNU Parallel y Crontab


Pre-requisitos.
Linux
Nodejs

Requisitos.
Casperjs
GNU Parallel

Instrucciones:
1.- git clone
2.- cd CasperjsParallel
3.- mkdir scraped_data
4.- cd ..
5.- ./scrap.sh

Y listo deberias a comezar a ver los archivos en el folder scraped_data.
